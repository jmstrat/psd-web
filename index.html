Help

Proof of concept: This tool is an early-stage prototype and has not been extensively tested. Please verify all outputs carefully. Whilst file type support is currently limited, the underlying calculation is universal and not technique-specific.

Once your data are loaded and analysed, the main PSD chart will display the phase-resolved data.

  • Zoom: Click and drag horizontally on the x-axis (double-click to reset).
  • View Profiles: Click any point to render its phase profile, which shows intensity as a function of the phase angle.
  • Select Angles: The point of maximum intensity renders as a single phase by default. To select a different angle manually, click anywhere on the phase profile.
  • Export: When you are happy with the results, click the "Download Results" button to save your data.

Requirements

  • Data should be provided as individual files (one per time point)
  • Data are assumed to be acquired at a fixed interval specified by “Acquisition Interval”
  • The first file must be at time 0 for the first cycle
  • File names must be named such that they increase numerically with time
  • An incomplete final cycle is ignored
  • The Cycle period does not need to be a multiple of the acquisition time, although this is preferable. Data will be re-binned if not.

Supported Files

  • .gr (PDFGetX3)
  • .xy (2-column data)
